# qa_python
test_add_new_book_add_two_books: Этот тест проверяет, что при добавлении двух книг в коллекцию, метод get_books_genre() возвращает список длиной 2.

test_add_new_book_add_one_books: Этот тест проверяет, что при добавлении одной книги в коллекцию, метод get_books_genre() возвращает список длиной 1.

test_add_new_book_add_books_more_then_42_letters: Этот тест проверяет, что при добавлении книги с названием длиннее 42 символов, книга не добавляется в коллекцию.

test_add_new_book_add_books_with_0_letters: Этот тест проверяет, что при добавлении книги с пустым названием, книга не добавляется в коллекцию.

test_add_new_book_add_two_the_same_books: Этот тест проверяет, что при добавлении двух книг с одинаковыми названиями, в коллекции остается только одна книга.

test_books_for_children_no_adult_genre: Этот тест проверяет, что если в коллекции есть книги для детей, то они не имеют жанра, который попадает под возрастной рейтинг для взрослых.

test_add_new_book_without_genre: Этот тест проверяет, что книга без указанного жанра возвращается со значением жанра по умолчанию (пустая строка).

test_get_list_of_favorites_books: Этот тест проверяет правильность работы метода get_list_of_favorites_books(), который должен возвращать список избранных книг.

test_delete_book_from_favorites: Этот тест проверяет удаление книги из списка избранных.

test_delete_non_existing_book_from_favorites: Этот тест проверяет, что удаление несуществующей книги из списка избранных не влияет на список.

test_delete_book_from_favorites (с параметрами): Этот тест также проверяет удаление книги из списка избранных с использованием параметризации для нескольких сценариев.
